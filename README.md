# Grade Book Manager

## Overview
Grade Book Manager is a Java console application that manages student grades across multiple sections. It allows users to enter grades with built-in input validation and provides a variety of grade analysis features, including highest and lowest marks, averages, passing and failing students, and individual student lookups.

The project demonstrates modular programming through well-structured methods, reusable input validation, and clean code organization.

## Features
- Manage multiple sections and students
- Input validation for menu selections, student counts, and grades
- Find the highest mark in a section
- Find the highest mark overall
- Find the lowest mark in a section
- Find the lowest mark overall
- Calculate the average for a section
- Calculate the overall average
- Display passing students
- Display failing students
- Display all student grades in a section
- Display an individual student's grade
- Modular method-based design
- Clean console-based interface

## How to Run
1. Ensure you have Java JDK installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/abdelhadibarakat/grade-book-manager.git
   ```
3. Navigate to the project folder:
   ```bash
   cd grade-book-manager
   ```
4. Compile the program:
   ```bash
   javac GradeBookManager.java
   ```
5. Run the program:
   ```bash
   java GradeBookManager
   ```
